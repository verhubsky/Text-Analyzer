# Text-Analyzer v0.1

## Обновление:
Логика валидации вынесены в отдельный блок. Проект разделен на модули.

## 📘 1. Краткое Описание:

### Простой скрипт Python, направленный на анализ текста и вывод специфической информации о нём:

1) количество слов;
2) количество уникальных слов;
3) топ-5 частых слов;
4) самое длинное слово;
5) среднюю длину слова.

## 🚀 2. Установка

```bash
git clone https://github.com/verhubsky/Text-Analyzer.git
```

## ⚙️ 3. Запуск

Перейдите в папку Text-Analyzer

```bash
cd Text-Analyzer
```

Запустите скрипт main.py

```bash
python main.py
```

(!) Потребуется ввести строку, которую скрипт будет анализировать. 

Пример работы скрипта:

```text
Enter words, for ex. 'I love cookies'
>>> I love cookies
-----------------------
| Text-Analyzer v0.1  |
-----------------------
1) Words: 3
2) Unique: 3
3) Top-5:
 1. I (1)
 2. love (1)
 3. cookies (1)
4) Longest: cookies
5) Average length: 4

```
